- 技能提升
    ```
    [skill data up]  
	`[职业]`	技能编号	`[适用地图]`	`[技能数据]`	编号	`提升类型`	提升数值  
    [/skill data up]
    ```
    范例：
    - 魔法师，47号技能，地下城中，等级变化属性，10号属性，百分比提升，提升14%
        ```
        [skill data up]
            `[mage]`	47	`[dungeon type]`	`[level]`	10	`%`	14	
        [/skill data up]
        ```
    - 男魔法师，3号技能，地下城中，固定数值属性，0号属性，百分比提升，提升18%
        ```
        [skill data up]
        `[at mage]`	3	`[dungeon type]`	`[static]`	0	`%`	18	
        [/skill data up]
        ```
